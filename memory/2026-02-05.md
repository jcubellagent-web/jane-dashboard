## Technical Notes

### ffmpeg Limitations
- Homebrew ffmpeg lacks drawtext/subtitles filters
- Solution: Use Pillow for frame-by-frame text overlay
- Process: Extract frames ‚Üí add overlays ‚Üí encode with audio

### Video Editing Workflow
1. Extract frames: `ffmpeg -i input.mp4 -q:v 2 frames/frame_%05d.jpg`
2. Process with Pillow (add text overlays)
3. Extract audio: `ffmpeg -i input.mp4 -vn -acodec copy audio.aac`
4. Recombine: `ffmpeg -framerate FPS -i frames/%05d.jpg -i audio.aac -c:v libx264 output.mp4`
